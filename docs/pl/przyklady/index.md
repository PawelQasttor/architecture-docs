# Przegląd Przykładów

Rzeczywiste przykłady demonstrujące Standard Dokumentacji Architektonicznej w praktyce.

## Kompletne Projekty

### Budynek Mieszkalny "Zielony Taras"

Kompleksowy przykład pokazujący kompletną dokumentację dla 6-piętrowego budynku mieszkalnego w Warszawie.

**Szczegóły Projektu:**
- **Typ:** Budynek mieszkalny wielorodzinny
- **Lokalizacja:** Warszawa, Polska
- **Wysokość:** 18.5m (6 kondygnacji)
- **Powierzchnia:** 1,800 m²
- **Lokale:** 18 mieszkań
- **LOD:** 400 (Dokumentacja budowlana)

**Co zawiera:**
- Kompletna specyfikacja projektu
- Specyfikacje elementów (ściany, stropy, okna)
- Specyfikacje materiałów
- Weryfikacja zgodności
- Integracja BIM (IFC 4.0)

[Poznaj projekt Zielony Taras →](/pl/przyklady/zielony-taras/)

*(Pełna polska wersja -- planowane)*

## Dlaczego Przykłady Są Ważne

Przykłady demonstrują:

✅ **Jak strukturyzować dokumentację**
- Poprawny frontmatter YAML
- Organizacja treści
- Wzajemne odniesienia

✅ **Rzeczywista weryfikacja zgodności**
- Zgodność z Prawem budowlanym
- WT 2021 cieplno/pożar/dostępność
- Rzeczywiste obliczenia i dane testowe

✅ **Integracja BIM w praktyce**
- Mapowania klas IFC
- Zestawy właściwości (Psets)
- Specyfikacje LOD 400
- Gotowe do synchronizacji dwukierunkowej

✅ **Jakość profesjonalna**
- Dokumentacja gotowa do budowy
- Gotowa do wniosków o pozwolenie
- Gotowa do eksportu PDF

## Wykorzystanie Przykładów

### Do Nauki

1. Przestudiuj strukturę
2. Zobacz jak odwołują się do przepisów
3. Zrozum integrację BIM
4. Naucz się najlepszych praktyk dokumentacji

### Dla Swoich Projektów

1. Skopiuj przykładowe pliki
2. Dostosuj do swojego projektu
3. Zaktualizuj frontmatter
4. Zmodyfikuj sekcje treści
5. Eksportuj do PDF

### Dla Zespołów

- Standaryzuj dokumentację
- Szkolenie nowych członków zespołu
- Wzorce jakości
- Szablony zgodności
