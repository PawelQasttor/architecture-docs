# Dokumentowanie istniejących budynków

Ten przewodnik wyjaśnia, jak stosować standard dokumentacji przy inwentaryzacji i opisywaniu istniejących budynków. Niezależnie od tego, czy pracujesz nad remontem, zmianą sposobu użytkowania, konserwacją zabytków czy zarządzaniem obiektem, standard zapewnia spójne ramy do rejestrowania stanu zastanego.

## Kiedy korzystać z tego przewodnika

Przewodnik ma zastosowanie zawsze, gdy dokumentujesz budynek, który już istnieje:

- **Remont i modernizacja** -- wymiana instalacji, wykończenia, zmiana układu pomieszczeń
- **Zmiana sposobu użytkowania** -- adaptacja budynku do nowej funkcji (np. magazyn na biura)
- **Zabytki i konserwacja** -- inwentaryzacja historycznej substancji i planowanie interwencji
- **Zarządzanie obiektem** -- stworzenie bazowej dokumentacji do bieżącej eksploatacji
- **Audyt przedtransakcyjny** -- przeglądy techniczne i ocena stanu przed zakupem

::: info Kluczowa różnica wobec nowych projektów
W przypadku istniejących budynków zaczynasz od [fazy Powykonawczej (Faza 7)](/pl/fazy/powykonawcza), a nie od Fazy 1. Najpierw rejestrujesz stan zastany, potem planujesz zmiany.
:::

## Punkt wyjścia: Książka Obiektu Budowlanego

W polskim systemie prawnym każdy budynek powinien posiadać Książkę Obiektu Budowlanego. Jest to cenne źródło informacji na starcie:

1. Sprawdź, czy książka obiektu istnieje i jest aktualna.
2. Wynotuj daty przeglądów okresowych, napraw i przebudów.
3. Zidentyfikuj brakujące wpisy lub luki w dokumentacji.
4. Wykorzystaj zebrane dane jako podstawę do uzupełnienia Karty Budynku.

::: tip
Nawet jeśli Książka Obiektu Budowlanego jest niekompletna, stanowi punkt odniesienia do porównania z wynikami inwentaryzacji terenowej.
:::

## Przebieg inwentaryzacji

Postępuj zgodnie z trzyetapowym schematem, aby systematycznie uchwycić stan istniejący.

### Etap 1: Pomiary

1. Zgromadź dostępną dokumentację archiwalną (rysunki oryginalne, pozwolenia, wcześniejsze inwentaryzacje).
2. Przeprowadź pomiary budynku kondygnacja po kondygnacji.
3. Zarejestruj osie konstrukcyjne, grubości ścian, wysokości międzykondygnacyjne i wymiary otworów.
4. W przypadku złożonych geometrii lub obiektów zabytkowych zastosuj skaning laserowy lub fotogrametrię.
5. Odnotuj rozbieżności między dokumentacją archiwalną a stanem faktycznym.

### Etap 2: Dokumentacja fotograficzna

1. Sfotografuj każde pomieszczenie z co najmniej dwóch przeciwległych narożników.
2. Utrwal wszystkie elewacje, stan pokrycia dachowego i kontekst otoczenia.
3. Udokumentuj widoczne uszkodzenia: pęknięcia, zawilgocenia, degradację materiałów.
4. Sfotografuj elementy ukryte tam, gdzie dostęp jest możliwy (przestrzeń nad sufitem podwieszanym, pod posadzką, w szachtach).
5. Stosuj spójną konwencję nazewnictwa plików, wiążącą zdjęcia z pomieszczeniami.

### Etap 3: Opracowanie dokumentacji

1. Sporządź rysunki inwentaryzacyjne na podstawie danych pomiarowych.
2. Utwórz karty dokumentacyjne opisane poniżej.
3. Zarejestruj niepewności i obszary wymagające dalszych badań.
4. Opracuj raport z inwentaryzacji podsumowujący ustalenia i jakość danych.

## Tworzenie kart dokumentacyjnych na podstawie inwentaryzacji

### Karta Budynku

Kartę Budynku tworzysz w pierwszej kolejności. Uzupełnij ją o:

| Pole | Źródło danych |
|---|---|
| Adres i współrzędne | Wizja lokalna, mapy |
| Data budowy | Księga wieczysta, Książka Obiektu, archiwa |
| Układ konstrukcyjny | Oględziny, dokumentacja oryginalna |
| Liczba kondygnacji | Pomiary inwentaryzacyjne |
| Powierzchnia całkowita brutto | Pomiary inwentaryzacyjne |
| Aktualna klasa funkcji | Obserwacja na miejscu, decyzje administracyjne |
| Ocena stanu technicznego | Wyniki przeglądu |

### Karty Kondygnacji

Utwórz Kartę Kondygnacji dla każdego poziomu, włącznie z piwnicami i poddaszem, jeśli jest dostępne:

1. Przypisz czytelny identyfikator (np. Kondygnacja -1, Kondygnacja 0, Kondygnacja 1).
2. Zapisz wysokość międzykondygnacyjną i rzędną posadzki względem przyjętego reperu.
3. Odnotuj typ konstrukcji stropowej i system nośny.
4. Dołącz odpowiedni rysunek rzutu kondygnacji.

### Karty Pomieszczeń

Utwórz Kartę Pomieszczenia dla każdego zinwentaryzowanego pomieszczenia lub strefy funkcjonalnej:

1. Przypisz unikalny numer pomieszczenia zgodny z przyjętą konwencją nazewnictwa.
2. Zapisz wymiary z pomiaru: długość, szerokość, powierzchnię, wysokość w świetle.
3. Udokumentuj aktualną funkcję i wykończenie (posadzka, ściany, sufit).
4. Odnotuj nietypowe warunki (spadki posadzki, nieregularna geometria, ograniczenia obciążenia).

### Karty Instalacji i Karty Urządzeń

Udokumentuj systemy techniczne budynku i kluczowe wyposażenie:

1. **Karty Instalacji**: ogrzewanie i wentylacja, instalacja elektryczna, wod.-kan., ochrona ppoż., transport pionowy.
2. **Karty Urządzeń**: kotły, agregaty chłodnicze, centrale wentylacyjne, rozdzielnice, dźwigi -- każde urządzenie wymagające konserwacji lub posiadające wartość odtworzeniową.
3. Zapisz producenta, model, wiek (jeśli znany), stan techniczny i wydajność.

Szczegółowe procedury zarządzania urządzeniami opisuje rozdział [Eksploatacja](/pl/eksploatacja/).

## Zakres informacji do zebrania

Poniższa tabela określa, jakie informacje należy zebrać dla każdego typu karty podczas inwentaryzacji budynku.

| Typ karty | Wymagane informacje | Typowe źródła |
|---|---|---|
| Karta Budynku | Adres, data budowy, konstrukcja, powierzchnia, kondygnacje, stan | Księga wieczysta, archiwa, pomiary |
| Karta Kondygnacji | Wysokość, rzędna, typ stropu, system nośny | Pomiary inwentaryzacyjne |
| Karta Pomieszczenia | Wymiary, funkcja, wykończenie, stan | Pomiary, dokumentacja fotograficzna |
| Karta Instalacji | Typ, wiek, wydajność, stan, zgodność z przepisami | Oględziny, dane z tabliczek znamionowych |
| Karta Urządzenia | Producent, model, numer seryjny, data montażu, stan | Tabliczki znamionowe, dzienniki konserwacji |

## Typowe wyzwania

::: warning Niekompletne informacje
Istniejące budynki rzadko dysponują pełną dokumentacją. Jasno określaj, co jest potwierdzone, co jest założeniem, a czego nie udało się ustalić. Stosuj ocenę wiarygodności danych (wysoka / średnia / niska) na każdej karcie.
:::

| Wyzwanie | Strategia postępowania |
|---|---|
| Brak dokumentacji oryginalnej | Pełna inwentaryzacja pomiarowa; kwerenda w archiwach urzędu miasta |
| Ukryta konstrukcja | Rewizje przez otwory inspekcyjne; badania nieniszczące (georadar, termografia) |
| Nieudokumentowane przeróbki | Porównanie pomiarów z dostępnymi pozwoleniami; wywiady z użytkownikami |
| Materiały niebezpieczne | Zlecenie badań specjalistycznych (azbest, ołów) przed pracami inwazyjnymi |
| Brak dokumentacji serwisowej | Rejestracja widocznych urządzeń; oznaczenie luk do dalszej weryfikacji |
