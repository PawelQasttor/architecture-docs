## Your path forward

Building documentation today is scattered across AutoCAD, Excel, Word and
email — and it never agrees with itself. SBM puts every room, zone, system
and asset in one plain-text file that humans and computers both read, and
a compiler turns those files into compliance reports, BIM parameters and
asset registers. **[Read the full rationale → What is SBM?](/en/standards/introduction)**
