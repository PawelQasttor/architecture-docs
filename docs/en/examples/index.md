# Examples Overview

Real-world examples demonstrating the Architecture Documentation Standard in practice.

## Complete Projects

### Green Terrace Residential Building

A comprehensive example showing complete documentation for a 6-story residential building in Warsaw.

**Project Details:**
- **Type:** Multi-family residential
- **Location:** Warsaw, Poland
- **Height:** 18.5m (6 stories)
- **Area:** 1,800 m²
- **Units:** 18 apartments
- **LOD:** 400 (Construction Documentation)

**What's Included:**
- Complete project specification
- Element specifications (walls, slabs, windows)
- Material specifications
- Compliance verification
- BIM integration (IFC 4.0)

## Why Examples Matter

Examples demonstrate:

✅ **How to structure documentation**
- Proper YAML frontmatter
- Content organization
- Cross-referencing

✅ **Real compliance verification**
- Prawo budowlane compliance
- WT 2021 thermal/fire/accessibility
- Actual calculations and test data

✅ **BIM integration in practice**
- IFC entity mappings
- Property sets (Psets)
- LOD 400 specifications
- Bi-directional sync ready

✅ **Professional quality**
- Construction-ready documentation
- Ready for building permits
- PDF export ready
